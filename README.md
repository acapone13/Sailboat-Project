Sailboat - AI Modules
===========================

Make sure you have [Python 3.6.x](https://www.python.org/downloads/ "Python 3.6.x") installed.

### Running the Project

##### Main Application:
    
  ```bash
  python3 src/main.py
  ```
    
##### Specific Tests:
    
  ```bash
  python3 src/tests/{desiredTest}.py
  ```
